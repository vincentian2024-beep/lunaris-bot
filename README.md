# Lunaris Bot

A Discord.js v14 bot for Lunaris Craft.

## Features

- Welcome System
- Slash Commands
- Discord.js v14
- Railway Deployment

## Setup

1. Create a Discord Bot in the Discord Developer Portal.
2. Copy your bot token.
3. Add the token to Railway:

DISCORD_TOKEN=your_token_here

4. Deploy to Railway.

## Run Locally

```bash
npm install
npm start
